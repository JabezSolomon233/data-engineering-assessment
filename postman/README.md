# Data Engineering Fresher Assessment

## Overview
This project contains:
- A FastAPI backend connected to PostgreSQL
- CRUD APIs for Users, Bank Info, and Employment Info
- An ETL script that merges and aggregates user data
- Postman collection with API test cases

## Tech Stack
- Python 3.14
- FastAPI
- PostgreSQL
- SQLAlchemy ORM
- Pydantic
- Uvicorn
- Pandas
